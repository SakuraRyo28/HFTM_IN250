# Markdown Basics

## Formatierung Titel

- ->  #    <- Haupttitel
- ->  ##   <- Untertitel
- ->  ###  <- Unter-Unter-Titel

## Textformatierung

- ->              ** s **             <-  Fett
- ->               * s *              <-  Kursiv
- ->              `Code`              <-  als Code
- ->  ``` mehrzeiliger Codeblick ```  <- in mehreren Zeilen 
- ->           > Quote                <- Block Quote (Zitat)

## Listen

### Geordnete Liste
- -> 1. <-
- -> 2. <-
- -> 3. <-

### Ungeordnete Liste
- -> - Eins <-
- -> - Zwei <-

### Tasklists
- -> [x] <- Erledigter Task
- -> []  <- Unerledigter Task
